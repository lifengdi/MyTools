# MyTools  
我的常用工具包，主要包括高精度的四则运算、时间日期计算和转换、字符串工具类；以及阿拉伯数字转中文

## ArithmeticUtil
- com.lifengdi.util 

此类实现了高精度四则运算

## DateTimeUtil
- com.lifengdi.util

时间日期工具类

## StringUtil 
- com.lifengdi.util

常用字符串工具类

## NumberToChinese
- com.lifengdi.change.impl

阿拉伯数字转中文数字，对外只提供了change(Long num)方法。

该类继承了AbstractNumChange类。

## AbstractNumChange
- com.lifengdi.change

数组转换抽象类。

此类实现了IChange接口。
